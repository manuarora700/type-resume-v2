# Type Resume

[View Site](http://www.manuarora.me/type)

## Building

```bash
git clone git@github.com:manuarora700/type-resume-v2.git
cd type-resume-v2
npm install
npm run dev
# Open localhost:4003/index-dev.html in your browser
```

## Building for Production

```bash
npm run build
```
